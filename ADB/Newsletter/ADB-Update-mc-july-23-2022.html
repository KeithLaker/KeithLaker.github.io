MULTI-CLOUD MANAGEMENT


INTEGRATION WITH AZURE

New - Deploy and manage Autonomous Database using Microsoft Azure


MULTI-CLOUD MANAGEMENT

CHALLENGE
Microsoft Azure customers want to simplify the deployment and management of their Autonomous Database instances by using their familiar Azure control plane.

SOLUTION
The Oracle Database Service for Azure portal provides a streamlined experience for establishing low latency OCI-Azure connectivity, virtual networks, and federated identity management.

Autonomous Database queries and query results are made possible through the Interconnect. The database logs and metrics from OCI are integrated into Azure to simplify monitoring and management.
