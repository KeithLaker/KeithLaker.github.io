ROADMAP


PLANNED NEW FEATURES COMING SOON

ADB-Shared feature roadmap for next 6-9 months


COMING SOON

Here is a sneak peek into features coming to Autonomous Database within the next 6-9 calendar months. The list of features is shown in alphabetic order - not release data order!

  • SIMPLER MANAGEMENT
    • dbms_cloud to support Google principal authentication
    • Flashback Data Archive
  • BETTER AVAILABILITY
    • Flashback Data Archive
    • Flexible maintenance windows
    • Long term backup retention
    • Restore terminated databases from backup
  • STRONGER SECURITY
    • Azure AD support
    • DB links to ADB-Shared with TLS (without a wallet)
    • Kerberos support
    • Wallet rotation with a grace period
  • SMARTER DATA and ANALYTICS
    • Data Lake Accelerator/Object Store Smart Scan
      • Currently in limited-availability
      • Improved scalability for data lake queries
      • More information here: https://create-new-url/adb-datalake
