AUTONOMOUS DATABASE
FACT 2
Autonomous Database keeps storage costs low and transparent

A Snowflake table consists of a series of micro-partitions where each one is 16MB in size. Snowflake's architecture means that updates force the creation of new micro-partitions during each update. The old, out-of-date data micro-partitions are not removed but continue to consume storage space. Snowflake customers can experience rapid storage bloat when doing even small-scale updates to their data warehouse.

Due to its robust architecture, Autonomous Database does not suffer from storage bloat during an update process. Autonomous Database customers need up to 31x less storage space in real-world scenarios.

AUTONOMOUS DATABASE
FACT 3
Autonomous Database processes real-world workloads up to 23x faster

Since Snowflake can only create new micro-partitions, there are no internal mechanisms to track and update existing data points efficiently. Snowflake believes that if existing data needs to be updated, it is always quicker to delete the existing data and completely reload the updated data. This approach has significant drawbacks in terms of performance and ease of use.

Autonomous Database has Oracles industry-leading database technology at its core, which contains a very sophisticated data management engine. Establishing business rules to enforce data integrity creates database structures that also help to speed up updates. Autonomous Database can process real-world update operations up to 23x faster.

FACT 4
Autonomous Database ensures data is always 100% trustworthy

Snowflake only supports the most basic data integrity rule - preventing null or empty rows from being loaded. It is possible for dimension tables and fact tables to contain duplicate rows, making it impossible to trust data shown in reports or charts fully.

Oracle Autonomous Database provides the most comprehensive range of data constraints and business rules covering unique rows and relationships between columns in different tables to ensure data is entirely trustworthy.
