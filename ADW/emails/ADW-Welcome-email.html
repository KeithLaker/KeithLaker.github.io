Tips to get you started

Your Autonomous Data Warehouse has a built-in suite of web-based tools called Oracle Database Actions.

Launch Database Actions

You can log in using the database admin username/password which you entered during database provisioning. These tools will help you do the following:


1

LOAD DATA          

Data Load tool provides wizards to help you load or access data from local files, from cloud storage, or from remote databases without writing a single line of code.

2

TRANSFORM DATA

Data Transform tool provides an easy to use range of transformations and prebuilt connectors. Simply assemble and configure sophisticated transformations which are highly performant and scalable.

DISCOVER PATTERNS

Data Insights tool displays information about patterns and anomalies in the data of entities in your Oracle Autonomous Data Warehouse.

EXPLORE YOUR DATA

Catalog tool provides information about the entities in your Autonomous Data Warehouse were created, and to understand the impact that changing the source of an entity may have on other entities.
