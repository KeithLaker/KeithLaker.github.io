Tips to get you started

Start working right away using the built-in suite of web-based tools called Oracle Database Actions.

Launch
Database Actions

You can log in using the database admin username/password which you entered during database provisioning. These tools will help you do the following:


1

LOAD DATA          

Data Load tool provides wizards to help you load or access data from local files, from cloud storage, or from remote databases without writing a single line of code. Click here to learn more.



2

TRANSFORM DATA

Data Transform tool provides an easy to use range of transformations and pre-built connectors. Simply assemble and configure sophisticated transformations which are highly performant and scalable. Click here to learn more.



3

BUSINESS MODEL

Business models open new avenues for analyzing data such as discovering patterns using Data Insights, improved hierarchy navigation, and the addition of hierarchy-aware calculations. Click here to learn more.



4

DISCOVER PATTERNS

Data Insights tool displays information about patterns and anomalies in the data of entities in your Oracle Autonomous Data Warehouse. Click here to learn more.



5

EXPLORE YOUR DATA

Catalog tool provides information about the entities in your Autonomous Data Warehouse were created, and to understand the impact that changing the source of an entity may have on other entities. Click here to learn more..

Tips to help you do more...

Your Autonomous Data Warehouse has a broad range of analytical features that can help you get more from your data. The following links will help you explore some of these more sophisticated capabilities:

VISUALIZE DATA

Oracle Analytics Desktop is free to use with your ADB. It helps you create reports and build interactive visualizations so you can quickly gain insight into your data. Click here to learn more.



MACHINE LEARNING

Oracle Machine Learning Notebooks is a built-in browser-based interactive data analysis environment that is part of Autonomous Database. It is designed for data scientists who want use the extensive library of in-database machine learning features. Click here to learn more.



GRAPH

Using Oracle Graph you can work on a wide range of business problems such as fraud detection in banking, find connections and link to data, and improve traceability in smart manufacturing. Click here to learn more.



SPATIAL

Oracle Spatial is included in Autonomous Database and it allows you to get started quickly and easily with location intelligence analytics and mapping services. Click here to learn more.
