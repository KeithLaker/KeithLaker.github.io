A history of data warehousing possibilities...

           
     
1977

Birth of a Silicon Valley startup

Engineers Larry Ellison, Bob Miner, and Ed Oates found Software Development Laboratories. Their first office is in Santa Clara, California.

           
     

SQL Arrives

First commercially available SQL-based RDBMS implementing some basic SQL queries and simple joins

           

Data Consistency

Multiversion read consistency is launched - data returned by a query is now committed and consistent with respect to a single point in time.

1984
      
           
     
1988

Online Backup and Recovery

An online (hot) backup is performed while the database is online and available for read/write operations - not interruption to service.

     

PL/SQL Arrives

Developers can now use a procedural language designed specifically to embrace SQL statements within its syntax.

     

Oracle Parallel Server

Software and a collection of hardware known as a "cluster", unites the processing power of each component to become a single, robust computing environment. A cluster generally comprises two or more computers, or "nodes".

           

PL/SQL Stored Procedure

Developers can now create reusable units of code that encapsulate specific business logic within data transformation workflows.

1992
      

Cost-Based Optimizer

The optimizer's job is to determine the best execution plan—the fastest way to get the desired result. The optimizer’s choice is based on the expected cost of execution. The optimizer balances the cost of taking each alternative.

     
           
     
1994

Parallel SQL Execution

Dramatically reduces response time for data-intensive operations on large databases typically associated with a data warehouse.

           

Partitioning Added

A powerful features that allows tables to be subdivided into smaller pieces, enabling them to be managed and accessed at a finer level of granularity. Result is much faster queries

1997
