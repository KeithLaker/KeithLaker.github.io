Pattern For Replacing Spread-Marts



Using spreadsheets for business critical functions is a disaster waiting to happen!

Almost every organization utilizes spreadsheets to disseminate and report on data. A far better solution is to have a single source of truth stored in a fully secured database with a browser-based app that everyone can use to maintain the data.

Let's Get You Started....


Stage 1 - SOURCE

Spreadsheet Files

Spreadsheets are so easy to create - anyone can put together a spreadsheet once they have the data. This is why spreadsheets are everywhere inside every organization. Once created they often send it out to colleagues to help update, and so forth.

This inevitably leads to numerous copies with different data and very flawed business processes. A far better solution is to have a single source of truth stored in a fully secured database with a browser-based app that everyone can use to maintain the data.

Stage 2 - CREATE

Oracle Autonomous Database

Oracle Autonomous Data Warehouse provides an easy-to-use, fully autonomous database that scales elastically, delivers fast query performance and requires no database administration.

Fully-managed cloud service that makes it very simple to provision a data warehouse, quickly and easily load data and query that data using built-in web-based tools such as notebooks.

Stage 3 - LOAD and DESIGN

Oracle Application Express

The easiest way to load spreadsheet data into Autonomous Data Warehouse is to use the file import wizard in Oracle Application Express. This then gives you the option to create an interactive report over that data.

Once the spreadsheet data has been loaded then that dataset can be shared with and accessed by any other BI reporting tool such as Oracle Analytics Desktop and Oracle Analytics Cloud.

Stage 4 - SECURING DATA

Oracle Data Safe

Data Safe is a unified control center for your Oracle Databases which helps you understand the sensitivity of your data, evaluate risks to data, mask sensitive data, implement and monitor security controls, assess user security, monitor user activity, and address data security compliance requirements. Whether you’re using Oracle Autonomous Database or Oracle Database Cloud Service (Exadata, Virtual Machine, or Bare Metal), Data Safe delivers essential data security capabilities as a service on Oracle Cloud Infrastructure.
