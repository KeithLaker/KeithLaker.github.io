Information About Managing APEX Service

You can access and manage APEX Service instances using Oracle Cloud Infrastructure (OCI) management console (sign-in link). In the console's service navigation menu, open Database Related Services, and click APEX Application Development. Here you can:

  • List APEX Service instances
  • Create a new APEX Service instance
  • Launch into APEX or Database Actions on an instance
  • Drill into instance details including APEX usage metrics
  • Access details of the underlying Autonomous Database

If you need to automate lifecycle management of APEX Service instances, try the Autonomous Database REST APIs. A similar interface is available for CLI.
