Oracle Global Leaders - Strategic Customer Program for Autonomous Database

The Oracle Global Leaders Strategic Customer Program is our global program to support projects using Autonomous Database. It is managed by the product management and development teams and provides expert-led solution planning and implementation reviews.





Program Overview


The Oracle Strategic Customer Program for Autonomous Database on Shared Exadata Infrastructure is designed to support customers who are just getting started with Autonomous Database and are looking for expert-led advice and guidance on how to successfully design and rollout their project.

There are many benefits in being part of this global program, including:

  • Ensure successful migration and implementation of existing workloads on Autonomous Database
  • Support from both product management and development
  • Dedicated coordinators for each geographical region
  • Program support for a project is targeted to last approximately three months but can be extended
  • Invitation to join the Oracle Global Leaders Program after successful implementation
  • Program is delivered using an honor system â€“ no contracts!

Services Offered


The Oracle Strategic Customer Program for Autonomous Database on Shared Exadata Infrastructure offers a number of key services to support Autonomous Database projects, including:

  • Advisory assistance through the development period and initial phase of project deployment
  • Weekly or Bi-weekly calls with ADB-S Product Management to review progress and best practices
    • Feature integration
    • Best practices for Autonomous Database
    • Project design review and verification
    • Initial project deployment to production
  • Checkup service to validate deployment and configuration
  • Invitation to join the Oracle Database Development Feedback Sessions
  • Invitation to join the Oracle Global Leaders regional and global meetings
