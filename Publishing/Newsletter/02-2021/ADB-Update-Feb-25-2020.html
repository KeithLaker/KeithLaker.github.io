GRAPH STUDIO FOR ADB
(Shared Infrastructure Only)
LIMITED AVAILABILITY - Graph Studio built-in to Autonomous Database

Graph console is now built-in to Autonomous Database. There is a launch button on the Tools panel of the ADB console. This integration provides comprehensive tooling for working with graphs:

  • Graph modeling tool to map relational data to graphs
  • Browser-based notebooks for interactive analysis and collaboration
  • Integrated graph visualization
  • Uses the proven capabilities of the Graph Analytics Engine

GA for this feature is expected around mid-March.

This is an important differentiating feature for data warehouse projects where the competition is Snowflake because Snowflake does not support graphs.

This is an important differentiating feature for data warehouse projects where the competition is Amazon, Microsoft and Google. These vendors rely on separate engines to manage graph data which means moving data from the warehouse to the graph engine and back again which adds latency and increases data security risks.

PYTHON
(Shared Infrastructure Only)
NEW - Oracle Machine Learning for Python

We are pleased to announce the general availability of the Oracle Machine Learning for Python on Autonomous Database.

Data scientists and other Python users can create user-defined Python functions managed in the database. Python objects can also be stored directly in the database – as opposed to being managed in flat files.

These features facilitate collaboration across the data science team by enabling convenient hand-off of data science work products from data scientists to application developers for immediate deployment.

OML4Py also supports automated machine learning—or AutoML—which not only enhances data scientist productivity, but also enables non-experts to use and benefit from machine learning. AutoML can help produce more accurate models faster, through automated algorithm and feature selection, and model tuning and selection.

PRIVATE ENDPOINTS
(Shared Infrastructure Only)
NEW - Private Endpoint support for built-in tools

ADB within a private endpoint now supports access to APEX, SQL Developer Web, ORDS, and OML Notebooks.

To connect Oracle Analytics Cloud instance to Autonomous Database that has a private endpoint, use the Data Gateway as per an on-premises database. See Configure and Register Data Gateway for Data Visualization for more information.
