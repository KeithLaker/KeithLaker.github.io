PEOPLESOFT + ADB
(Shared Infrastructure)
PeopleSoft Now Certified On Autonomous Database

Oracle PeopleSoft announced certification of PeopleSoft 9.2 products on OCI Autonomous Database – Shared. To run Oracle PeopleSoft 9.2 on OCI ADB-S, following are the required minimum PeopleTools versions and patch levels:

  • PeopleTools 8.57 Jan ’21 CPU (Patch 20)
  • PeopleTools 8.58 Jan ’21 CPU (Patch 09)
  • PeopleTools 8.59 (GA planned this week) – latest version of PeopleTools.

Note: Certification for Oracle PeopleSoft 9.2 products on OCI Autonomous Database – Dedicated was announced last year in July.
